## Current UX Inventory

### Pages That Exist
1. **`project-builder.html`** - 5-step wizard (generic)
2. **`catalog.html`** - Product grid with filters
3. **`cart.html`** - Shopping cart
4. **`product-detail.html`** - Individual product PDP
5. **`account.html`** - User account area
6. **`order-history.html`** - Past orders

### Key Components
- **Wizard Progress:** Horizontal step indicator with circles
- **Photo Tiles:** Large photo cards with overlay text (Steps 1-2)
- **Icon Tiles:** Icon + text cards (Steps 3-4)
- **Kit Sidebar:** Flyout sidebar with items list (during "Kit Mode")
- **Bundle Display:** Table-based product list (Step 5)
- **Filters Sidebar:** Left-side catalog filters

### Problems to Eliminate

#### 1. **"Kit Mode" - REMOVE THIS CONCEPT**
- **Problem:** Confusing mental model, unclear when you're "in" or "out" of it
- **Solution:** Replace with persona-specific flows (Marcus's "Customize Materials", Lisa's "Add Optional Items")

#### 2. **Generic Wizard for All Users**
- **Problem:** Sarah (production builder) shouldn't see the same UI as David (DIY homeowner)
- **Solution:** Purpose-built experiences per persona

#### 3. **Step 5 Identity Crisis**
- **Problem:** Is it a PDP? A review step? A shopping mode launcher?
- **Solution:** Make it the **Kit PDP** for some flows, but Sarah doesn't use wizards at all
